## GCC

### Compilar

> #### Compilado normal
>     gcc loop.c -o loop
>
> #### Incluir información de depuración (debugging)
>     gcc -g loop.c -o loop
>
> #### Compilar para arquitectura específica
>     gcc variables.c -m32 -o variables-32
